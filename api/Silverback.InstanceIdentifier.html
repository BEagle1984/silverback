    Class InstanceIdentifier

    The identifier used to distinguish the instances of the same type. Used mostly for logging and debugging.

    Remarks

    Using this class instead of a System.Guid allocates more memory (about 152 bytes, instead of the 16 bytes used by a System.Guid) because the value is stored directly as string to avoid extra allocations in the ToString() method (e.g. when writing to a log).

    This is also a reference type and passing is around requires less allocations.

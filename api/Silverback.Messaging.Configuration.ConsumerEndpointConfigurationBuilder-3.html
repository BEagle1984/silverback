Specifies that the subscribers can subscribe a stream (IAsyncEnumerable<T>, IEnumerable<T>, etc.) pushed with all messages consumed from that endpoint (or partition). This setup can be used for basic stream processing, but it's more fragile and should be used with care. Some features, like error policies, won't work as expected. This is disabled by default.

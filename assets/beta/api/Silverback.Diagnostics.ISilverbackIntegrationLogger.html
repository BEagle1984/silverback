    Interface ISilverbackIntegrationLogger

    Extends the ISilverbackLogger<TCategoryName> adding some methods used to consistently enrich the log entry with the information about the message(s) being produced or consumed.

    Assembly: cs.temp.dll.dll
    Syntax
